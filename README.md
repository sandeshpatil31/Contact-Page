# React Contact Form

A functional and responsive contact page built with **React**, featuring controlled components and state management.

## 🚀 Features
- **Controlled Components**: Uses `useState` for real-time input tracking.
- **Form Validation**: Checks for empty fields and valid email patterns before submission.
- **Loading & Success States**: Visual feedback during and after form submission.
- **Responsive UI**: Styled for all screen sizes.
